# Sistema de Gerenciamento de Estoque

Um sistema de gerenciamento de estoque desenvolvido em **Java** com o objetivo de praticar os principais conceitos da Programação Orientada a Objetos (POO) e manipulação de arquivos.

O projeto implementa um **CRUD (Create, Read, Update e Delete)** de produtos através de um menu interativo executado no terminal.

---

# Funcionalidades

- Cadastro de produtos
- Listagem de todos os produtos cadastrados
- Edição de produtos
- Remoção de produtos
- Persistência dos dados em arquivo `.txt`
- Menu interativo no terminal

---

# Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- Arrays
- Encapsulamento
- Scanner
- Manipulação de Arquivos (`File` e `FileWriter`)

---

# 🗄 Armazenamento dos dados

Este projeto **não utiliza banco de dados**.

Como o objetivo é praticar os fundamentos da linguagem Java antes de trabalhar com tecnologias como **MySQL**, **PostgreSQL** ou outros bancos de dados, foi utilizado um **Array** como um banco de dados improvisado.

Cada posição dos vetores representa um produto cadastrado.

Além disso, sempre que uma alteração é realizada (cadastro, edição ou exclusão), os dados também são gravados em um arquivo `.txt`, funcionando como uma persistência simples das informações.
